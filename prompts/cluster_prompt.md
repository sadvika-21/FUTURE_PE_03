# Content Cluster Prompt

## Objective

Generate SEO content clusters for an Organic Grocery Store website.

---

## Role

Act as an SEO Content Strategist.

---

## Prompt

Generate one pillar page and multiple supporting blog topics.

The content cluster should improve website authority and internal linking.

---

## Pillar Page

Organic Grocery Store Guide

---

## Supporting Topics

Generate articles related to:

* Organic Fruits
* Organic Vegetables
* Healthy Snacks
* Dairy Products
* Sustainable Living
* Eco-friendly Shopping
* Seasonal Produce
* Meal Planning
* Healthy Recipes
* Organic Farming

---

## Output Format

Provide:

* Pillar Topic
* Cluster Topic
* Target Keyword
* Search Intent
* Suggested Internal Link

---

## Goal

Create a content cluster strategy that improves SEO performance and enhances user navigation across the website.
